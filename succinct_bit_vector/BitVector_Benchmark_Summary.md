# Bit Vector Benchmark Results

## Полные результаты производительности

### Основные операции (100K данных)

| Библиотека          | Rank (ns/op) | Select (ns/op) | Access (ns/op) | Сложность Rank | Сложность Select |
|---------------------|--------------|----------------|----------------|----------------|------------------|
| **RSDic**           | 31.74        | 95.65          | 24.51          | O(1)           | O(log n)         |
| **RankDirectory**   | 95.17        | 1223           | -              | O(1)           | O(log n)         |
| **BitString Naive** | 614.4        | -              | 12.41          | O(n)           | -                |

*Ускорения: RSDic rank 19.4x быстрее BitString, RSDic select 13.1x быстрее RankDirectory*

### Масштабируемость операций (ns/op)

| Библиотека        | Операция | 1K    | 10K   | 100K  | 1M    | Деградация (1K→1M) |
|-------------------|----------|-------|-------|-------|-------|--------------------|
| **RSDic**         | Rank     | 24.09 | 26.67 | 27.89 | 30.76 | +27.7%             |
| **RSDic**         | Select   | 50.65 | 61.35 | 104.3 | 97.33 | +92.2%             |
| **RankDirectory** | Rank     | 97.71 | 144.6 | 152.7 | 161.5 | +65.3%             |
| **RankDirectory** | Select   | 1265  | 2071  | 2932  | 3542  | +180.0%            |

### Дополнительные операции RSDic

| Операция | Производительность | Описание        |
|----------|--------------------|-----------------|
| PushBack | 25.52 ns/op        | Добавление бита |
| Access   | 24.51 ns/op        | Доступ к биту   |

## Теоретические vs Экспериментальные результаты

| Операция       | Теоретическая сложность | Экспериментальные данные |
|----------------|-------------------------|--------------------------|
| RSDic Rank     | O(1)                    | 27.7% деградация (1K→1M) |
| RSDic Select   | O(log n)                | 1.92x замедление (1K→1M) |
| RankDir Rank   | O(1)                    | 65.3% деградация (1K→1M) |
| RankDir Select | O(log n)                | 2.80x замедление (1K→1M) |

*Результаты получены на Apple M4, Go 1.25.0*
